# Sales Performance Analysis Dashboard

## Overview

This project showcases an interactive Power BI dashboard designed to analyze sales performance across different regions, categories, and time periods.

## Key Features

* Total Revenue, Average Order Value, and Total Orders KPIs
* Sales breakdown by category and country
* Monthly revenue trends
* Interactive filters (Region, Category, Date Range)
